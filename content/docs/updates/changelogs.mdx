# Server Changelogs

Stay up to date with all the latest changes, updates, and improvements to the Legacy Roleplay Bangladesh server.

## December 22, 2021

### Patch Notes: Alpha 2.40

- Added 'clear file' parameter to /doors_scan development command.
- Added burgershot store at the drive through.
- Added /execute/setTime and /execute/setWeather REST API routes.
- Added key bind support in the settings for the '/e c' command.
- Added more chat emotes.
- Added pigeon milk and wonder waffles to the EMS store.
- Added streamer mode (Can be toggled via /streamer_mode).
- Added support for garages locked to jobs.
- Added Tic-Tac item.
- Balanced gun running drops.
- Disabled using the radio while down.
- Fixed on duty time tracking.
- Fixed oxy runs.
- Fixed Xenon headlight colors disappearing when the vehicle is repaired.
- IFAK now has the same effects as oxy.
- Removed damage and injuries from snowballs.
- Slowed down the natural alcohol drop off.
- Various balancing changes.
- General optimizations, fixes and improvements to the framework.

## December 15, 2021

### Patch Notes: Alpha 2.39

- API logs are now deleted after a week.
- Added '/vehcontrol'.
- Added a '/death_timer' command for super-admins.
- Added a '/debug_damage' command.
- Added a '/riot_mode' command for super-admins.
- Added a '/toggle_duty_status' command for developers.
- Added a 'LockerOverrides' config option that allows you to override which job belongs to which inventory lockers.
- Added a 'NeutralGangs' config option that will prevent gangs from attacking players when shooting.
- Added a 'PopulationDensityMultiplier' config option that lets you override the default population density multiplier.
- Added a PD locker at N.O.O.S.E.
- Added a basic 'PD' and 'hospital' to Cayo Perico.
- Added a better fingerprinting system.
- Added a clothing store at the casino.
- Added a garage at Cayo Perico.
- Added a lot of new modification options to the LSC.
- Added an access level '3' for the property system.
- Added config option 'DoorClusterKeyOverrides' that allows you to override which job belongs to which door clusters.
- Added more chat emotes.
- Added the ability to drive a boat to Cayo Perico.
- Added the ability to properly spawn at Cayo Perico.
- Added various stores to Cayo Perico.
- Disabled a bunch of commands for DOC.
- Evidence is less likely to become ruined now.
- Fixed 'christmas priority'.
- Fixed double purchasing of vehicles at PDM.
- Fixed fenders in the LSC.
- Fixed jerrycans giving a lot of stress when pouring gasoline and fire extinguishers when fire extinguishing.
- Fixed ped objects sometimes deleting nearby entities accidentally.
- Fixed the 'on duty time'.
- Fixed the 'server id' parameter not working on '/wallhack'.
- Fixed the Mission Row PD clothing spot.
- Fixed the advent calendar not showing for everyone.
- Fixed the casino blackjack logs.
- Fixes to walkstyles saving.
- Job stores will now show whether or not you can use them.
- Moved the 'bean machine' and 'burger shot' stores.
- Possibly fixed the 'infinite weather' issue with weather spells.
- Removed a lot of broken peds from the creation menu.
- The '/tp*' commands now work with vehicles.
- The '/wipe' command will now have a clearer warning when you're wiping the entire world.
- The weather on Cayo Perico will now always be sunny. Various balancing changes.
- Vehicle retrieval costs are now based on their inventory content weights.
- Vehicles are no longer automatically impounded on server stop.
- Vehicles in garages will now be sorted by when they were last pulled out.
- Weapon flashlights will now auto-toggle when aiming. When in the inventory and hovering over an item, clicking the mouse wheel will now attempt to use that item.
- When loading a character, the voice server will be automatically rejoined.
- You now only need 22 'bonus points' in the advent calendar to get the bonus.
- General optimizations, fixes and improvements to the framework.

**FORGOTTEN:**
- Vehicles that have not had their finance paid in the last 2 weeks will be automatically repoed.
