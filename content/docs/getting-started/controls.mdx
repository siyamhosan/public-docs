# Controls & Keybinds

Essential controls and commands for playing on Legacy Roleplay Bangladesh.

<Tabs items={['General', 'In-Vehicle', 'Commands']}>
  <Tab value="General">
  <>
  
### General Controls

The following keybinds are essential for gameplay:

- **P** = Phone
- **K** = Inventory
- **/emote** or **/e** = Animations
- **N** = Push-to-talk (default)
- **Y** = Change voice range
- **U** = See players online and player IDs
- **E** = Interact
- **F** = Enter/exit Vehicle
- **L** = Lock/unlock vehicle
- **R** = Reload
- **Ctrl** = Crouch
- **Z** = Prone
- **C** = Look behind you
- **V** = Toggle first/third person view
- **1-5** = Use items in your hot bar

#### Useful Commands:

- **/mask** = Toggle mask
- **/glasses** = Toggle glasses
- **/trunk** = Access vehicle inventory
</>
  </Tab>
  <Tab value="In-Vehicle">
  <>
### In-Vehicle Controls

Essential controls when driving:

- **H** = Headlights
- **E** = Horn
- **/engine** = Toggle engine
- **/door 1-6** = Open/close specific door
- **/window 1-6** = Roll windows up/down
- **Z** = Cruise Control
- **B** = Seat belt
</>
  </Tab>
  <Tab value="Commands">
  <>
### Chat Commands

Important commands for communication:

- **/ooc** = Out of character chat for technical problems
- **/oocl** = Local out of character for specific situations
- **/me** = Display an action (e.g., `/me performs CPR`)
- **/report** = Report a player or situation to staff (include player IDs)
</>
  </Tab>
</Tabs>

<Callout type="info">
  These keybinds can also be found on our Discord server. Some can be customized in settings.
</Callout>

## Tips

- **Practice** these controls before joining roleplay situations
- **Customize** keybinds in settings if needed
- **Use /report** if you encounter technical issues or rule violations
