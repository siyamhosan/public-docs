## Quick Reference

### Zero Tolerance Rules

These violations result in **permanent bans without appeal:**

<Callout type="error">
  **Critical Violations:**
</Callout>

- **VDM (Vehicle Death Match)** - Using vehicles as weapons
- **RDM (Random Death Match)** - Killing without proper roleplay
- **Combat Logging** - Disconnecting during active roleplay
- **Metagaming** - Using out-of-character information
- **Harassment/Discrimination** - Any form of harassment or discrimination

### Key Rules to Remember

- **Fear RP** must be shown in dangerous situations
- **New Life Rule (NLR)** applies after death - you forget the last 15 minutes
- **Quality roleplay** is required for all interactions
- **Group size limits** must be followed for criminal activities

## Rule Violations

If you experience rule violations:

<Steps>
  <Step>
    **Document the Incident**
    
    Take screenshots or recordings as evidence. Note down player IDs and timestamps.
  </Step>
  <Step>
    **Report In-Game**
    
    Use `/report` in-game with player IDs and a brief description.
  </Step>
  <Step>
    **File a Formal Report**
    
    If needed, file a [Report](/docs/community/applications#report-section) with all evidence included.
  </Step>
</Steps>
