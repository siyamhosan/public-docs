# Boosting Rules

Rules for vehicle boosting and police pursuits.

## Category 1: (Boosting Class D, C)

**Total Pursuing Unit:** 2  
**Allowed vehicles:** Only Vehicles like Caprice and Crown Vic, Explorer

## Category 2: (Boosting Class B, A)

**Total Pursuing Unit:** 3  
**Allowed vehicles:** All individual authorized vehicles

## Category 3: (Boosting Class M, M+, A+, S, S+)

**Total Pursuing Unit:** 4

**Allowed vehicles:** All individual authorized vehicles. 1 Motorcycles and 1 speed unit can be used.

## Rules for Officers

<Callout type="info">
  **Important Guidelines:**
</Callout>

- No air one will be used
- The suspect has to be pursued if he abandons the boosting car
- But the boosted car has to be secured as a priority
- Ensure the safety of the boosting car with one officer

## Rules for Criminals

<Callout type="warning">
  **Code Red Guidelines:**
</Callout>

- Don't do Code red unless its necessary and you have a solid RP behind it
- If you do code red without any reason just because you couldn't escape then It will be counted as Poor RP
- If you want to submit the car at drop-off location, you have to escape first from the PD officers
- Submitting cars in between chase will consider as Power Gaming

<Callout type="info">
  We are putting this rule to make fun for both parties!
</Callout>
