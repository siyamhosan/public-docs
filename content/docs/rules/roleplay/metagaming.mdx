# Metagaming

Metagaming is the act of using information obtained from outside of Roleplay and influencing Roleplay inside the game and is not permitted.

## (4.1) Definition

Metagaming is using information obtained from outside of roleplay to influence roleplay situations in-game.

<Callout type="error">
  **Metagaming is strictly prohibited and can result in permanent bans.**
</Callout>

### (4.1.1) Discord During RP

If you use Discord while RP is running or if any RP situation starts from Discord / another communication site, you can get BAN from server.

## (4.2) ID Identification

You can't identify someone by looking at their ID. You can only use this ID number when reporting to someone. If you identify someone with their ID, you can BAN from the server.

<Callout type="warning">
  Only use player IDs for reporting purposes, never to identify players in-game.
</Callout>

## (4.3) Communication Devices

If your communication devices (mobile, radio) have been removed from you, you must not use that device to communicate.

## (4.4) Stream Sniping

Getting any information by watching someone's stream and using that information in the game is also called Metagaming. If you do this, you can be banned from the server.

<Callout type="error">
  Watching streams to gain information about players or situations is metagaming and will result in a ban.
</Callout>

## Examples of Metagaming

- Using Discord/Voice chat during roleplay situations
- Identifying players by their ID number
- Using information from streams or outside sources
- Using removed communication devices
