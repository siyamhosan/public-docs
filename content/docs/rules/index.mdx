## Quick Reference

### Zero Tolerance Rules

<Callout type="error">
  **These violations result in permanent bans:**
</Callout>

- **VDM** - Using vehicles as weapons
- **RDM** - Killing without proper roleplay  
- **Combat Logging** - Disconnecting during roleplay
- **Metagaming** - Using outside information
- **Harassment/Discrimination** - Any form of harassment
